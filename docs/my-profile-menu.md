Recommended Profile Menu
------------------------

### Core (must have)

-   Profile
-   Subscription / Plan
-   Settings

* * * * *

### Training-related (high value)

-   My Stats (shortcut to progression)
-   Achievements / Badges
-   Rankings (global / continental position)

* * * * *

### Content / Personalization

-   My Drills
-   Saved Sessions / Favorites

* * * * *

### System / Utility

-   Notifications
-   Help / Support
-   Feedback / Report issue
-   About / Version

* * * * *

### Optional (only if needed)

-   Friends / Social (if you add it later)
-   Connected devices (if you add video / hardware)
-   Data export

* * * * *

What to avoid
-------------

-   Duplicates of main navigation (Stats, History, Calendar already exist)
-   Anything session-related (belongs in main flow)
