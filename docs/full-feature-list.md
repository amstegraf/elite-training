Features (concise list)
-----------------------

### Core (existing / baseline)

-   Session tracking (balls, pots, misses)
-   Miss categorization (alignment, delivery, speed, position)
-   Position outcome logging (playable / pot miss / no shot)
-   Rack-by-rack tracking
-   Basic stats (pot %, position %, conversion %)
-   History + calendar + progression charts

* * * * *

### Training & Performance

-   **Baseline onboarding (10 racks)**
-   **Session goals (focus per session)**
-   **Error distribution dashboard (% miss types)**
-   **Skill tiers (Amateur → Elite based on KPIs)**
-   **Pressure mode (timed / perfect racks / restart rules)**

* * * * *

### Drills System

-   Drills library (predefined)
-   Create drill manually
-   **Create drill from image (AI → JSON ball mapping)**
-   Save drills to profile
-   Run drills as sessions
-   Drill performance tracking

* * * * *

### Match Mode

-   2-player match tracking
-   Score + racks
-   Log misses during match
-   Match history
-   Basic opponent comparison

* * * * *

### AI Features (core differentiator)

-   **AI Coach (session report)**
-   **AI Coach (long-term stats analysis)**
-   **Drill generator (based on weaknesses)**
-   **Table image → ball mapping (JSON)**
-   Pattern suggestion (planned vs actual - optional later)
-   (Future) Video analysis

* * * * *

### Data & Insights

-   Trend analysis (1W / 1M / 3M / etc.)
-   KPI breakdown (pot / pos / conv weighted)
-   Consistency score (variance)
-   Streak tracking
-   Peak performance detection

* * * * *

### User & System

-   Player profiles (multi-user)
-   Session tagging (training / match / drill)
-   Notes per session
-   Export/share session report

* * * * *

Tiers
-----

### 🟢 Free

-   Unlimited sessions
-   Basic stats (pot / pos / conv)
-   Limited history (e.g. 7--14 days)
-   Manual drills only
-   Global ranking (Only after 50 recorded session)
-   Continental ranking

* * * * *

### 🟡 Pro

-   AI Coach (session + stats)
-   Full history in cloud
-   Advanced analytics (error distribution, trends, consistency)
-   Drill library + tracking
-   Drill path gamification
-   Tournaments and matches
-   Image → drill (AI table mapping)

* * * * *

### 🔴 Elite

-   Match mode
-   Pressure training modes
-   Drill generator (AI)
