# 🎱 Pool Training App – Player Guide

## 🧠 What This App Is

This application is a **performance-driven pool training system** designed to help players:

- Understand **why they miss**
- Track **real progress over time**
- Identify **true bottlenecks (not assumptions)**
- Improve **position play, consistency, and rack conversion**

It is not a scoring app.
It is a **diagnostic and improvement system**.

---

# 🎯 Core Concept

The app is built around one principle:

> Track failures → analyze causes → improve precision → increase consistency

Instead of tracking every shot, you only log:
- misses
- position breakdowns
- execution issues

This keeps the data **clean, fast to input, and highly relevant**.

---

# 🧩 Main Components

## 1. Dashboard

The dashboard gives a **real-time snapshot of your level** using 3 key KPIs:

### 🔴 Pot Success
- % of balls you successfully pot
- Measures **pure shot-making ability**

### 🟢 Position Outcome
- % of good cue ball results
- Measures **how often you land correctly for the next shot**

### 🔴 Rack Conversion
- % of racks you fully complete
- Measures **real match performance**

---

## 🧠 How to Read It

- High Pot + Low Position → you miss patterns, not shots
- High Position + Low Conversion → you fail under pressure
- Balanced values → stable player

---

## 2. Training Program

You train inside structured programs (e.g. "Precision – 90 days")

Each program contains:
- Phases (e.g. 20cm → 10cm precision)
- Modes (Rack / Drill)
- Table configuration

---

## 3. Session Tracking (Ghost / Rack Mode)

During a session:

- You play normally
- You ONLY log when something goes wrong

### Miss Types
- Position
- Speed
- Alignment
- Delivery

### Outcomes
- Playable (suboptimal)
- Pot miss
- No shot (position breakdown)

---

## 🧠 Why this works

- No friction during play
- Focus on meaningful events only
- Captures real weaknesses

---

## 4. Session Report

After each session, you get a full breakdown:

### 🎯 Performance
- Avg balls per rack
- Best run
- True miss rate
- Rack conversion

### 🔁 Flow & Resilience
- Flow efficiency
- Recovery rate
- Failed recoveries

### 🎯 Execution Matrix
- Pot success
- Position outcome

### 🔍 Failure Breakdown
- Position vs Speed vs Alignment

---

## 🧠 What this tells you

- Where you fail (pattern, execution, pressure)
- How severe your errors are
- Whether you recover or collapse

---

## 5. Advanced Analytics

### 📈 Rack Consistency
- Shows variance between worst and best racks
- Measures **stability**

### 📊 Per Rack Analysis
- Balls cleared per rack
- Miss distribution

### ⚖️ Position vs Speed Trends
- Shows whether errors come from:
  - wrong direction (position)
  - wrong magnitude (speed)

# 📊 Player Levels (Based on KPIs)

Your level is determined by:
- Potting

- Amateur → pots but cannot run racks
- Strong Amateur → starts converting (~10%)
- Advanced → stable patterns (~35%)
- Semi-pro → consistent execution (~50%)
- Elite → high precision + conversion (~65%+)
