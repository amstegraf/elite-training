What it does

Precision pool training: programs and plans, live sessions with rack-by-rack logging (misses, outcomes, tags), session reports, progress charts, and optional Electron desktop packaging that runs the same FastAPI app locally.

  • Session data as JSON under data/sessions/ (override with ELITE_TRAINING_DATA_DIR).
  • Programs and plans in data/programs.json.
  • Tier matrix and thresholds configurable under Settings → Training tiers.

Documentation

Product direction and metrics are described in the repo under docs/ (for example docs/training-platform-description.md, docs/metric-position-success.md, docs/matrix-calculation-of-tier.md).
